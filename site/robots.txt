# Crawl policy for narenana.com
#
# Open to crawlers by default. This covers three surfaces on this host:
#   /            the landing page (main website)
#   /log-viewer/ the EdgeTX / iNAV / Betaflight log viewer — a separate
#                Cloudflare Pages app proxied through this Worker
#   /wings/      the RC-plane catalog (category, landing and product pages)
# The password-gated admin panel and its JSON API are disallowed — they carry
# no crawl value and 401 anyway.
#
# Sitemaps: the primary one lists the landing page, /log-viewer/ and every
# /wings/ page. The Nanawing FPV simulator runs on its own subdomain
# (sim.narenana.com) and ships its own sitemap, referenced below so this file
# is a complete index of the property.

User-agent: *
Allow: /
Disallow: /admin
Disallow: /api/

Sitemap: https://www.narenana.com/sitemap.xml
Sitemap: https://sim.narenana.com/sitemap.xml
